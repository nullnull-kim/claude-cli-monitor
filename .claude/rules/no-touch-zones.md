# No-touch zones

> Pulled at session start and before any file write outside `src/`/`test/`. Referenced from CLAUDE.md "Rules Index".

These paths must not be modified by any agent without explicit user authorization in the current session. "I think the user would want this" is not authorization.

## Hard-deny paths

| Path | Why | If you need to change | 
|------|-----|----------------------|
| `node_modules/**` | Vendor dependencies. Reproducible from `package.json`. | Modify `package.json` and run `npm install`. Never hand-edit. |
| `node_modules/claude-diet/**` | Runtime hook dependency for the diet feature. **NOT this project's code.** Its CLAUDE.md and agent definitions are for *that* package, not us. | Open an issue upstream. Do not patch in place. |
| `dist/**` | TypeScript build output. Regenerated by `npm run build`. | Edit the `src/` source. |
| `*.tsbuildinfo` | TS incremental build cache. Regenerated automatically. | Delete to force rebuild — never edit. |
| `package-lock.json` | npm-managed lockfile. | Run `npm install` or `npm update`. Never hand-edit. |
| `~/.claude-agent-monitor/state/**` | Runtime state written by the project's own state.ts at runtime. | Modify only via `state.ts` functions. Never write directly. |
| `~/.claude-mem/**` | claude-mem internal storage (SQLite + Chroma). | Use claude-mem CLI / MCP tools. Never write directly. |
| `~/.claude/projects/**/memory/**` | Auto-memory store (`MEMORY.md` + per-topic `.md` files). Owned by Claude itself per the system memory protocol. | Update through normal memory write protocol (read MEMORY.md, write a topic file, link from MEMORY.md). Do not bulk-edit or delete. |
| `.git/**` | Git metadata. | Use `git` commands only. Never edit refs/objects directly. |
| `study/**` | Korean translation / reference mirror. Read-by-human only, **never read by agents**. Do not Read, Glob, or Grep into this directory during normal work. | Update only when the user asks for translation refresh. |

## Soft-deny paths (read-only by default; ask before edit)

| Path | Why |
|------|-----|
| `LICENSE` | Legal text. Only changes on license switches. |
| `CHANGELOG.md` (if present) | Updated only on releases. |
| `.github/workflows/**` | CI/CD. Changes touch shared infra. |
| `.claude/agents/**` | Agent definitions are stable contracts. Edit only when an agent's responsibilities change. |
| `.claude/settings.local.json` | Per-developer permissions. Each developer owns their own copy. |
| `C:\Users\kimty\CLAUDE\claude-code\**` | Reverse-engineered Claude Code reference source. Reference only — never patch. |

## Other-project boundary

This project lives at `C:\Users\kimty\CLAUDE\_projects\claude_agent_monitor`. Sibling projects under `C:\Users\kimty\CLAUDE\_projects\` are out of scope.

- Do not read other projects' files unless the user explicitly references them.
- Do not modify other projects' files ever.
- Do not apply other projects' CLAUDE.md rules (e.g. `node_modules/claude-diet/CLAUDE.md` belongs to that package, not us).
